# 🍽️ Питание СПБ

Система управления заказами питания для организаций Санкт-Петербурга.

## О приложении

**Питание СПБ** — это веб-приложение для:

- 📋 Оформления заказов на питание
- 👥 Управления клиентами (договоры, скидки, менеджеры)
- 📱 Отправки уведомлений в Telegram
- 📊 Админ-панели для менеджеров


### Возможности

- **Клиентский личный кабинет**
  - Просмотр меню и цен
  - Оформление заказов
  - Отправка сообщений менеджеру
  - История заказов

- **Админ-панель**
  - Управление клиентами
  - Просмотр и подтверждение заказов
  - Настройка скидок и условий
  - Тестирование Telegram

- **Интеграции**
  - Telegram бот для уведомлений
  - Google Таблицы для базы данных

## 📖 Инструкция по использованию


### Вход в систему

1. Открой приложение
2. Введи номер договора (например: `TEST-001`, `TEST-002`)
3. Нажми **Войти**


### Тестовые аккаунты

| Договор | Компания | Тип |
|---------|----------|-----|
| `TEST-001` | ООО ТехноСтрой | Офис |
| `TEST-002` | ИП Сидоров | Производство |
| `DEMO` | Демо клиент | Офис |
| `ADMIN` | Админ-панель | — |

### Как сделать заказ

1. Войди в систему с договором `TEST-001`
2. Перейди на вкладку **📋 Заказы**
3. Выбери день недели
4. Выбери приёмы пищи (завтрак, обед, ужин)
5. Нажми **Отправить на согласование**

Заказ появится в админ-панели для подтверждения.


### Админ-панель

1. Войди с договором `ADMIN`
2. Там ты увидишь:
   - **Клиенты** — список всех клиентов
   - **Заказы** — история заказов
   - **Ожидание** — заказы на подтверждение
   - **Telegram** — настройки и тест

## ⚙️ Настройка Telegram

### Получение токена бота

1. Открой @BotFather в Telegram
2. Создай нового бота: `/newbot`
3. Скопируй токен (пример: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)


### Получение ID группы

1. Добавь бота в группу
2. Отправь сообщение в группу
3. Перейди по ссылке: `https://api.telegram.org/bot<TOKEN>/getUpdates`
4. Найди `chat.id` (будет вида: `-1001234567890`)


### Настройка в коде

Отредактируй файл `src/App.jsx`:

```javascript
const BOT_TOKEN = 'ТВОЙ_ТОКЕН';
const GROUP_CHAT_ID = 'ТВОЙ_ID';
```

### ID топиков (для форумов)

```javascript
const THREADS = {
  waiting: 360,   // Ожидание заказов
  newUser: 361,   // Новые пользователи
  history: 362,   // История
```

## 🛠 Техническая информация

### Стек

- **Frontend**: React + Vite
- **Стили**: Inline CSS
- **Хранение**: LocalStorage + Google Таблицы
- **Деплой**: Vercel / Netlify

### Структура проекта

```
food-production/
├── src/
│   ├── App.jsx      # Основное приложение
│   ├── main.jsx     # Точка входа
│   ├── index.css    # Стили
│   └── routes.jsx   # Маршруты (если есть)
├── index.html      # HTML шаблон
├── package.json    # Зависимости
└── vite.config.js  # Конфигурация Vite
```

### Переменные

- `SPREADSHEET_ID` — ID Google Таблицы
- `BOT_TOKEN` — Токен Telegram бота
- `GROUP_CHAT_ID` — ID группы Telegram
